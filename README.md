# cheat-sheets

## Basic Git Commands
- copy repository
```
git clone [url]
```
- show status of files in repository
```
git status
```
- prepare file for next commit
```
git add [filename]
```
- commit changes
```
git commit -m "[commit message]"
```
- update files with changes from commit
```
git push
```
- get changes and update local repository
```
git pull
```
